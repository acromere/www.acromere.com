The Acromere website

# Front End

The front end is a React-based application that provides the user interface of 
the Acromere website.

## Development

To run the front end locally, follow these steps:

1. Clone the repository
2. Install dependencies with `npm install`
3. Start the development server with `npm start`

# Back End

The back end is a Java-based application that provides the core functionality of 
the Acromere website. It mainly supports the Xenon repository capabilities.
